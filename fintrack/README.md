# FinTrack

## Sobre
O FinTrack é um programa em Java feito para ajudar no controle de gastos pessoais pelo terminal.

## Problema
Muitas pessoas não controlam seus gastos e acabam perdendo dinheiro.

## Solução
O sistema permite registrar e acompanhar os gastos de forma simples.

## Funcionalidades
- Adicionar gasto
- Listar gastos
- Remover gasto
- Ver total gasto

## Como usar
1. Baixe o projeto
2. Abra no Eclipse
3. Execute o arquivo principal
